화면 구성

앱의 전체 레이아웃

상단 헤더
날짜 이동 버튼 / 팀 배지 / 취소·다시 버튼 / 운행·휴차 범례 / 다크모드 토글(왼쪽 상단)
배차 현황 (관리자 전용)
헤더 날짜 기준 오늘·내일 배차 버스 번호 표시. 파란색=총5회차(조기완료), 흰색=총6회차, 노란색=휴차
주차 그리드
2R~7R(+추가 행) 각 6칸씩 구성. 흰색=운행, 노란색=휴차, 빈칸=미배정
게시판
전체·공지·메모 탭으로 구분. 특이사항 메모 및 공지 확인
하단 메뉴
메모 / 로그인 버튼 (관리자 로그인 시: 차량·행 / 팝업 / 메모 / 설정 / 로그아웃)
🚌

배차 현황

오늘 · 내일 배차 버스 번호 확인 (관리자 전용)

⬜ 흰색 = 총6회차 (일반)
🔵 파란색 = 총5회차 (조기완료)
🟡 노란색 = 휴차
사용 방법
1
관리자 로그인 후 배차 현황 섹션 확인
로그인하면 주차 그리드 위에 배차 현황 섹션이 나타납니다
2
불러오기 버튼 탭
불러오기 버튼을 누르면 API에서 오늘·내일 배차 데이터를 가져와 Firebase에 저장합니다
3
날짜 변경 시 자동 연동
헤더 날짜를 바꾸면 해당 날짜에 저장된 배차 데이터를 자동으로 불러옵니다. 저장된 데이터가 없으면 불러오기 버튼 안내가 표시됩니다
💡 배차 데이터는 2일 지나면 자동 삭제됩니다. 불러온 데이터는 날짜별로 저장되어 같은 날짜 재접속 시 즉시 표시됩니다.
🅿️

주차칸 설정

운행·휴차 배치 및 상태 관리

⬜ 흰색 = 운행 중
🟡 노란색 = 당일 휴차
□ 빈칸 = 미배정
관리자 전용 기능
운행 ↔ 휴차 전환 관리자
주차칸을 꾹누르면(0.5초 이상) 운행↔휴차 상태가 전환됩니다
차량 위치 이동 관리자
칸을 탭해 선택(파란 테두리)한 뒤 다른 칸을 탭하면 위치가 바뀝니다.
↩ 취소 / ↪ 다시 관리자
헤더의 취소/다시 버튼으로 최근 변경을 되돌릴 수 있습니다 (날짜별로 독립 관리)
🔐

관리자 기능

로그인 후 사용 가능

1
관리자 로그인
하단 로그인 버튼 → 비밀번호 입력 → 하단 메뉴 전체 활성화
2
차량·행 수정
차량·행 버튼 1회 탭 = 차량 수정 모드 / 2회 탭 = 행 수정 모드 / 3회 탭 = 종료. 수정 후 ✅ 수정 완료 버튼으로 저장
3
팝업 설정
특정 요일·시간에 자동 안내 팝업 표시. 팝업1, 팝업2 등 여러 개 설정 가능
4
앱 설정
권한 제어 / 팀 표시 방식 / 하단 안내문 수정 가능. 저장 시 모든 방문자에게 즉시 반영
⚠️ 각조 팀장 & 부팀장 허락 없이 주차도 수정 절대 금지
📅

날짜 이동

원하는 날짜의 주차도 확인

1
‹ › 버튼
하루씩 이전/다음 날짜로 이동합니다
2
날짜 직접 선택
가운데 날짜를 탭하면 달력이 열립니다. 원하는 날짜를 선택하세요
3
오늘 버튼
오른쪽 검정 버튼을 누르면 오늘 날짜로 즉시 이동합니다
💡 날짜가 저장된 적 없으면 주차칸이 흐릿하게 표시됩니다. 관리자가 배치를 저장하면 정상 표시됩니다.
📋

게시판

메모 · 공지 · 댓글

탭 구분
전체 — 최신 활동순 / 📢 공지 — 공지글만 / 메모 — 일반 메모만
글쓰기
하단 메모 버튼 탭 → 내용 입력 → 등록. 관리자는 📢 공지글 체크박스로 공지 등록 가능
펼치기 / 접기
게시글은 기본 1줄 요약. ▼ 버튼으로 전체 내용을 펼칠 수 있습니다
댓글
댓글 N개 ▼ 버튼으로 펼치기. 가장 최신 댓글이 위에 표시됩니다
페이지
게시글이 많으면 하단 페이지 번호로 이동합니다 (페이지당 5개)
⚙️

개인 설정

내 기기에만 적용되는 설정

다크모드
헤더 왼쪽 상단 ☀️/🌙 버튼으로 전환. 내 기기에만 적용되며 재접속해도 유지됩니다
앱 설치 (PWA)
처음 접속 시 하단에 설치 안내가 표시됩니다. 설치하면 홈 화면에서 앱처럼 바로 실행 가능합니다

홈 화면에 추가

앱처럼 설치하기

Android (크롬)
접속 후 하단 설치 배너에서 설치 버튼을 탭하세요
iPhone / iPad (사파리)
사파리에서 접속 → 하단 공유 버튼(□↑) 탭 → "홈 화면에 추가" 선택
💡 설치 후에는 인터넷 연결만 있으면 앱처럼 전체화면으로 이용할 수 있습니다
