기본 사용

  • 날짜 이동 < > / 날짜 / 오늘
  • 주차 상태 흰색(운행) / 노란색(휴차) / 빈칸
🔐

관리자

  • 로그인 하단 +버튼
  • 주차 변경 꾹 누르기(0.5초이상)-> 노란색 변경
  • 위치 이동 번호선택(파란테두리) → 원하는 위치탭이동
  • 취소/다시 ↩취소 - 다시 ↪
  • 번호 표시 오늘/내일 동일 → 빨간 테두리
🚌

배차

  • 로그인 후 + 메뉴
  • 불러오기 오늘/내일 차량 번호 데이터 순서대로,
⚙️

기타

  • 차량/행 수정 1·2·3회 탭 설정
  • 팝업 설정 시간별 안내
  • 다크모드 ☀️/🌙
  • 홈화면 추가 공유(□↑)
